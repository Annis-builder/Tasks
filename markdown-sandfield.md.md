##markdown学习笔记  
  
一.在官网--[这个链接](https://markdown.com.cn/basic-syntax/) 翻看基本语法教学文件.然后编辑  

二.以下是总结  

1. 标题

标题及级别区分

```markdown
# 一级标题
## 二级标题
### 三级标题
```

2. 强调

```
*斜体文本* 或 _斜体文本_
**粗体文本** 或 __粗体文本__
***粗斜体*** 或 ___粗斜体___
```

3. 列表

无序列表：使用 -, *, 或 +

```
- 项目一
- 项目二
  - 子项目
```

有序列表：使用数字加点

```
1. 第一项
2. 第二项
```

4. 链接与图片

· 链接：[链接文字](网址)
· 图片：![图片描述](图片网址)

5. 代码

· 行内代码：用一个反引号包裹  `代码` 
· 代码块：用三个反引号包裹，并可指定语言以实现语法高亮
  ```markdown
  ```python
  print("Hello, World!")
  ```

6. 引用

使用 > 号。

```markdown
> 这是一个引用块。  
>>这是两个.
```

7. 分割线

三个或更多的 - 或 *。

```
---
```
